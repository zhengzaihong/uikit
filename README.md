
# flutter 常用组件ui库
 1. 可自定义日历组件
 2. 输入框搜索组件
 3. 进度条组件(条形和圆形)
 4. 评分组件
 5. 吐司提示,可完全自定义
 6. 灵活popwindow组件
 7. 城市picker
 8. 复选框，单选，气泡,抽屉组件等
 9. 倒计时控件
 10. 增加类似android TextView 上下左右可添加元素组件
 11. 添加可自定大小的switch组件
 
  ...持续更新中....
